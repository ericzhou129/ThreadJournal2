Implementation Notes

Audio Pipeline

  • AVAudioSession: Category: PlayAndRecord, allows background audio
  • AVAudioEngine: 44.1kHz sampling, mono channel, real-time buffer processing
  • Buffer Size: 1024 samples for low latency
  • VAD: 3-second silence threshold for auto-stop

WhisperKit Integration

  • Model: openai_whisper-small (39MB, multilingual)
  • Streaming: Partial results every 0.5 seconds
  • Performance: <1s to first partial on A16+ chips
  • Fallback: Queue for later if model not available

UI/UX Considerations

  • Recording Modes: Hold-to-record for short thoughts, tap-to-record for longer entries
  • Visual Feedback: Waveform animation and pulsing record button
  • Thread Suggestion: Show confidence score, allow manual override
  • Accessibility: Full VoiceOver support, haptic feedback

Privacy & Security

  • On-Device Only: No audio or transcriptions sent to servers
  • Model Storage: Private app container, not in backups
  • Permissions: Microphone access required, clearly explained
  • Data Retention: Audio buffers cleared immediately after transcription
